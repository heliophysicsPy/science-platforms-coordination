# Science Platforms Coordination HelioCloud PyHC Image
## (For Binder/JupyterHub)

The PyHC Environment Docker image containing all published PyHC packages, with added packages inspired by HelioCloud's base image, to be used by the Science Platforms Coordination IHDEA working group.
